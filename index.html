Oppsett — Google Cloud

  1. Gå til console.cloud.google.com
  2. Opprett et nytt prosjekt
  3. Aktiver Google Drive API og Google Docs API
  4. Gå til APIs & Services → Credentials
  5. Klikk Create Credentials → OAuth client ID
  6. Velg type: Web application
  7. Under Authorised JavaScript origins, legg til adressen appen kjører på (f.eks. http://localhost:8080 eller din HTTPS-domene)
  8. Kopier Client ID
  9. Åpne app.js og erstatt DIN_CLIENT_ID_HER med din Client ID
Merk: Appen lagrer kun filer den selv oppretter (begrenset tilgang). Den leser ikke andre filer i Google Drive.
